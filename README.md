# Task Manager

This is a simple task manager application built with Node.js.

## Features

-   Registration users
-   Login/logout users
-   Add new tasks
-   Edit existing tasks
-   Mark tasks as completed
-   Delete tasks
-   View all tasks
-   Filter tasks (all, completed, incomplete)

## Usage

### Install Dependencies

```
npm install
```

### Run Server

```
npm start
```

This will start the server on port 8002.
And connect to the database

## Technologies Used

-   Node.js
-   Express
-   Passport-js
-   MongoDB

## Technologies Used
